## 🩷 Pinkflix

Este proyecto es una pequeña aplicación web donde puedes buscar series de televisión y guardar tus favoritas.

Forma parte de la evaluación final del módulo 2 del bootcamp de programación de Adalab, y ha sido una forma de poner en práctica todo lo aprendido hasta ahora con JavaScript.

## 🔍 ¿Qué hace?

- Puedes buscar series usando la API de TVMaze
- Ver los resultados con su imagen y nombre
- Añadir series a favoritos haciendo click
- Eliminar favoritaws cuando quieras
- Evitar que se repitan
- Guardar los favoritos aunque recargues la página

También añadí un botón para limpiar todos los favoritos y un mensaje que aparece cuando la lista está vacía.

## 🛠️ Tecnologías

He utilizado:

- HTML
- SCSS
- JavaScript
- Vite
- API de TVMaze
- LocalStorage

## 🧠 Qué he aprendido

Sobre todo he trabajado:

- Cómo manejar eventos en JavaScript
- Cómo recorrer arrays y trabajar con ellos
- Cómo conectar con un API y usar los datos
- Cómo actualizar la interfaz dinámicamente
- Cómo guardar información en localStorage

También he intentado organizar el código para que sea más fácil de leer.

## 🎨 Diseño

He querido darle un estilo sencillo y limpio, con una paleta de rosas suaves y una estética bastante minimalista.

## 🚀 Cómo usarlo

1. Escribe el nombre de una serie
2. Haz click en una para añadirla a favoritos
3. Puedes eliminarla o limpiar toda la lista
4. Los cambios se guardan automáticamente
